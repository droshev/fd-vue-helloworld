# fd-vue-helloworld

## Project setup

```
npm install -g serve
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run it productively

```
serve -s dist
```

Source: https://cli.vuejs.org/guide/deployment.html#general-guidelines
